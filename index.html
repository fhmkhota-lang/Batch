The plan 05 / 06

Twelve months to prove it.

Months 1–3

Spring

Set up, R&D, develop flavours. Selling by month 3 — Uber Eats, direct, markets.

Months 4–6

Summer

Grow into the market. Test platforms, stockists and markets. Build the audience.

Months 7–9

Autumn

Learn where it can go — which channels work, where demand is.

Months 10–12

Winter

Assess sustainability. If proven — justify the store, with the audience already built.

The goal isn't profit yet. It's proof — demand, channels, and an audience for the store.
